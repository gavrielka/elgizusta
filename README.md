# Elgiz Usta - Peşəkar Telefon Təmiri Landing Page

A modern, high-converting landing page designed for **Elgiz Usta**, a professional phone technician service based in Baku (Metro Xalqlar Dostluğu).

## 🚀 Overview

This project features a premium "tech-repair" aesthetic built with standard web technologies. It is optimized for lead generation through direct calls and WhatsApp integration, specifically tailored for the Azerbaijani market.

## ✨ Features

- **Modern Dark UI**: A sleek, tech-focused design with neon accents and glassmorphism.
- **Conversion Focused**: Strategically placed CTA buttons for immediate phone calls and WhatsApp chats.
- **Fully Responsive**: Mobile-first design ensures a great experience on all devices.
- **Micro-interactions**: Scroll-triggered animations and dynamic stat counters.
- **Azerbaijani Language Support**: All copy is written in natural, professional Azerbaijani.

## 🛠️ Technology Stack

- **HTML5**: Semantic structure for SEO and accessibility.
- **Vanilla CSS**: Custom design system using CSS variables and modern flexbox/grid layouts.
- **JavaScript**: Lightweight interactivity and scroll effects.
- **Lucide Icons**: Modern and clean vector iconography.
